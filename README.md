## Greeting application

```bash
jdk - oraclejdk8
build - maven
log - logback
test - junit
```

####Console application that displays:

```bash
Good morning, World! if current time is 06:00 - 09:00
Good day, World! - 09:00 - 19:00
Good evening, World! - 19:00 - 23:00
Good night, World! - 23:00 - 06:00
```
